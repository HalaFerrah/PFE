# Frontend (React + Vite)

Boat assurance frontend application.

## Run

```bash
npm install
npm run dev
```

## Key pages

- `/home`: welcome page with boat visuals
- `/quote/duration`: step 1
- `/quote/boat`: step 2
- `/quote/guarantee`: step 3 with 3414 guarantee categories
- `/register`: account creation
- `/account`: user account page after registration
- `/payment`: payment continuation

## API endpoint

Set `VITE_API_URL` in `.env` (optional):

```env
VITE_API_URL=http://127.0.0.1:3000/api
```
